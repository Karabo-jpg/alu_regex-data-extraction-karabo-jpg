# Regex Data Extraction Project

## Overview
This project extracts specific types of data from text using **Regular Expressions (Regex)**.  
The script can identify and extract:

- Email addresses
- URLs
- Phone numbers (Ugandan & Kenyan)
- Credit card numbers
- Hashtags
- Time (24-hour format)
- HTML tags
- Currency (UGX)

## Requirements
- Python 3.x
